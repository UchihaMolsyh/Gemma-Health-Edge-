Common Issues

AI responses are slow or timing out

Check your backend server status. If using local mode, ensure llama-server is running. Try restarting the server or switching to a different AI backend.

PDF import is not working

Ensure the PDF contains selectable text (not scanned images). Try using a smaller PDF file under 10MB. Check that the PDF.js library loaded correctly.

Camera button not appearing

Enable the camera feature in Settings > Features. Ensure your browser has camera permissions granted. Some browsers require HTTPS for camera access.

Language not changing

After selecting a language, the change is automatic. Some interface elements may require a page refresh to fully update.
